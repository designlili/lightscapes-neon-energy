# Lightscapes – Neon Energy Generator

An interactive digital art project inspired by light, healing, movement and intuition.

This project transforms motion into glowing neon trails and particles, creating a unique visual experience between art, energy and technology.

## Features
- Interactive mouse movement
- Glowing neon particles
- Dark immersive atmosphere
- Inspired by healing art and light expression
